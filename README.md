# abp6_td
📊 Proyecto: Análisis Estadístico sobre Hábitos Saludables en Jóvenes Universitarios

📌 Descripción del proyecto

Este proyecto tiene como objetivo analizar los hábitos de sueño, actividad física, alimentación y niveles de estrés en jóvenes universitarios, utilizando herramientas de inferencia estadística. A través de la simulación de datos y el análisis estadístico, se busca identificar patrones relevantes que permitan comprender el comportamiento de esta población y apoyar la toma de decisiones en políticas de bienestar estudiantil

🎯 Objetivo

Desarrollar una investigación estadística completa aplicando el método científico, con el fin de obtener conclusiones válidas sobre los hábitos saludables en estudiantes universitarios (simulado).

🧠 Metodología

El proyecto se desarrolló siguiendo el método científico:

Planteamiento del problema
Se analizó la relación entre las horas de sueño y el nivel de actividad física en estudiantes.
Formulación de hipótesis
H0: Las horas de sueño no influyen en la actividad física.
H1: Las horas de sueño influyen en la actividad física.
Recolección de datos
Se generó un dataset simulado de 150 estudiantes utilizando Python y NumPy.
Análisis estadístico
Se aplicaron técnicas como:
Estadística descriptiva
Distribuciones de probabilidad
Teorema del Límite Central
Intervalos de confianza
Test de hipótesis
Conclusiones
Se interpretaron los resultados obtenidos para generar recomendaciones.
